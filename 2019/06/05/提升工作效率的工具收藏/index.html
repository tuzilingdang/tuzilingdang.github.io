提升工作效率的工具收藏

工欲善其事，必先利其器，好的工具要收藏一下。

终端工具

iTerm2 有美好配色的命令行工具

oh-my-zsh 简化复杂的命令

截图工具

snipaste 截图标记非常方便

FireShot 可滚动截屏的网页截图工具

全屏取色工具

snip 小而美的取色工具，可以建立起自己钟爱的配色方案了

博客工具

hexo 快速搭建个人博客

图床工具

Picgo 一键上传，让博客加图片变的简单

录屏及GIF制作工具

Kap 录屏神器，可以处理成gif格式

UML绘制工具

starUMl 垒代码前记得做好设计哟

思维导图

mindNode 极简风格，连官网都设计的好看极了

Git 图形化界面工具

gitkraken 界面非常炫酷，很Geek

sourcetree 已粉gitkraken

图片优化

tinypng

vscode插件

Turbo Console Log

一键插入想要打印的console.log

Image Preview

实时图片预览，非常方便

Local History

按时间的文件存储，防止代码丢失
